# Es_Par
~~~pseint
//Escribe un programa que pida un número al usuario y siga
//pidiendo hasta que el usuario ingrese un número par.
Algoritmo Es_Par
  num <- 0
  Repetir
  Escribir "Ingrese un numero: "
  Leer num
  Hasta Que num mod 2 = 0
  Escribir "El numero es par"
FinAlgoritmo
~~~
