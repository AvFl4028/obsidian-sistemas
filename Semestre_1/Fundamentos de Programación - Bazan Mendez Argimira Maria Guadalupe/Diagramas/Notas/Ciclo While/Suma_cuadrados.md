# Suma_cuadrados
~~~pseint
//Escribe un programa que pida números enteros positivos y sume
//sus cuadrados. El ciclo debe terminar cuando el usuario ingrese
//un número negativo y mostrar la suma total de los cuadrados.

Algoritmo Suma_cuadrados
  num <- 1
  sum <- 0

  Mientras num es mayor que 0 Hacer
  leer num
  sum <- sum + num^2
  FinMientras
  Escribir "La suma de los cuadrados es: ", sum
FinAlgoritmo
~~~
